# Платформа «Сбор работ на конкурс»

Веб-приложение для проведения конкурсов с разделением ролей (администратор, жюри, участник).

## Функциональность

- ✅ Регистрация и аутентификация пользователей
- ✅ Три роли: администратор, жюри, участник
- ✅ Управление конкурсами (администратор)
- ✅ Создание и редактирование подач (участник)
- ✅ Загрузка файлов (PDF, ZIP, PNG, JPG до 10МБ, макс. 3 файла)
- ✅ Комментирование подач
- ✅ Изменение статусов подач (жюри)
- ✅ Очереди для проверки файлов и уведомлений
- ✅ Управление пользователями и ролями (администратор)

## Технологии

- Laravel 11
- MySQL
- Очереди (database)
- Bootstrap-подобный CSS (кастомный)
- MinIO/S3 (опционально)

## Требования к окружению

- PHP 8.1+
- Composer
- MySQL (OpenServer, XAMPP или любой другой)
- Node.js (опционально, для сборки frontend)
